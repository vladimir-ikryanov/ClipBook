# Cross-Platform Clipboard Sync - Architecture Specification

## Project Overview

A secure, end-to-end encrypted clipboard synchronization application across macOS, Windows, Android, and iOS. Desktop apps provide full clipboard monitoring while mobile apps offer a simple paste-to-sync interface.
